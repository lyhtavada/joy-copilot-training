# Troubleshooting Template

## Symptom
Describe the observable symptom (what the user sees).

## Possible Causes
- Cause 1: Short explanation
- Cause 2: Short explanation

## Fix / Step-by-step Resolution
1. Quick checks: network, app version, account status.
2. Reproduce the issue and capture screenshots or logs.
3. Apply fixes: clear cache, force quit and re-open app, re-login.
4. Verify resolution with the user.

## When to Escalate
- If server errors or repeated failures after standard fixes.
- If issue affects multiple users or merchant integration.
- Required artifacts: user ID, app version, OS, timestamps, screenshots, request IDs.
