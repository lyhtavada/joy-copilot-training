### [hashtag](#how-to-create-a-workflow-between-okendo-and-joy-loyalty) How to create a workflow between Okendo and Joy Loyalty?

1

**Access Shopify Flow and Create a New Workflow**

- You need to access Shopify Flow and create a new workflow. Here, you should choose Okendo as the trigger

Shopify Flow

2

**Select Action**

- Select '**Action**'. Here, you should choose 'Joy loyalty' to automatically add points for the new review

Select Action

2

**Choose review app**

**Select Okendo** as the review app to integrate with
