### [hashtag](#what-is-sign-up-feature) What is "Sign up" feature?

**Sign up** feature allows you to reward new customers when they join your loyalty program. It's an automated way to welcome new members with points or discounts, encouraging immediate engagement with your store.

### [hashtag](#why-do-you-need-this-feature) Why do you need this feature?

**Sign-up rewards** are a practical way to attract new members to your loyalty program and keep them engaged from the start. In this guide, we'll cover the essentials: understanding the rules, setting it up, and following best practices to ensure your rewards program runs smoothly.

### [hashtag](#how-to-set-up-this-feature-on-your-store) How to set up this feature on your store

1

**Get access to Sign up**

Choose **Reward programs** -> **Earning programs** -> Click **Add rule** -> Choose **Sign up**

2

**Set up Sign up**

a. **Rule information**

- **Rule name**: Enter program name that appears on your widget and loyalty page
- **Start date** and **End date**: Set your program duration

b. **Reward rule**

Set up the reward that new members will receive by select **Choose reward**

- **Points:**

Reward customers with an amount of points

Select **Points** from reward type down and enter earning amount in **Earning points** section

- **Discount amount** and **Discount percentage:**

Reward customers with ready-to-go coupons

- **Customize discount codeâ€™s prefix:** Create your own code prefix
- **Applicable categories:** Define the scope of the discount code
- **Minimum order requirement:** Minimum amount of the order for the coupon to be valid
- **Coupon expiration date:** Set the expiration date for the discount code
- **Condition:** Configure whether this discount can be combined with other discounts and set the sales channels where this discount can be applied

3

**Save** and **Turn on** the Sign up
